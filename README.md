# Sales Data Analyzer

A beginner-friendly Python data analysis project built using pandas and matplotlib.

## What this project does

This project analyzes sales data from a CSV file and helps understand:

- product-wise sales
- category-wise sales
- daily sales
- missing values
- duplicate rows
- basic data cleaning
- chart visualization

## Features

- Read sales data from CSV
- Display dataset details
- Select and filter data
- Sort values
- Create a new `TotalSales` column
- Group data by product and category
- Find top product and top category
- Handle missing values
- Remove duplicate rows
- Create bar charts and line charts
- Save chart images to the `output` folder

## Technologies used

- Python
- pandas
- matplotlib
